🚀 OpenAPI Servers

Helm Repository for OpenAPI Server Tools

📦 Quick Start

Add this Helm repository and install the charts:

# Add the repository
helm repo add openapi-servers https://euh2.github.io/openapi-servers
helm repo update

# Install the chart
helm install my-api-tools openapi-servers/openapi-servers

# Install with custom values
helm install my-api-tools openapi-servers/openapi-servers -f values.yaml
